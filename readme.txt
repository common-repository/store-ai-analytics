== Description ==

Store AI Analytics & Reports is a flexible analytics solution built on WordPress. Now visualize and improve perfomance of your WooCommerce store with us. This plugin is developed by ClouDhiti.ai. Store AI Analytics & Reports uses analytics.cloudhiti.ai as a platform to perform Data Analytics and provides it's users, AI enriched dashboard. The base platform can be directly accessed here : https://analytics.cloudhiti.ai/login . Kindly read the Terms & Policy in the same page : https://analytics.cloudhiti.ai/login . The user data is also stored with analytics.cloudhiti.ai. 
ClouDhiti (meaning “Insights from the Cloud”) is the world’s first marketplace for AI Apps. These AI Apps bring the powers of Artificial Intelligence (AI) into the hands of Small & Medium sized Businesses (SMBs) - for their business good.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To auto-install Store AI Analytics, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “Store AI Analytics,” then click “Search Plugins.” Once you’ve found it,  you can view details about it such as the point release, rating, and description. Most importantly, you just need to click on  “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the Store AI Analytics plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Configure =

After installing and activating the plugin you will get a form to register with Store AI Analytics. ( If already registered, then click on Existing Users ).
To register fill in the form, click on Register and relax!"
